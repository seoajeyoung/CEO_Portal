💻프로그램 사용 설명서

1.아이디 파일, 설명 내용 파일은 메모장에 “ , ” 와 “enter” 로 구분해주시면 됩니다.

예시)
아이디1, 비밀번호1
아이디2, 비밀번호2

2.해시태그 파일은 메모장에 “ , ” 로 구분해주시면 됩니다.

예시) #맛집, #피자, #라면

3.내용 파일은 메모장에 “ / ” 로 구분해주시면 됩니다.

예시) 안녕하세요./오늘은 고기가 먹고싶어요/해외여행

4.이미지 폴더는 폴더명과 이미지명이 영어 이여야 합니다.

예시)image_folder 안 ex_image1.jpg, ex_image.png ...

폴더 안 이미지는 보이지 않습니다. 폴더 용량은 최대 50MB를 넘으면 안됩니다.

* 한번 사용한 이미지는 제외됩니다.


5.해쉬태그 개수를 입력하면 입력한 개수 사이에 해쉬태그가 추가됩니다.

예시)최소:5 최대:10 → 5~10개중 랜덤한 개수의 해쉬태그가 추가


6.게시할 갯수(아이디당) 아이디당 몇번 게시물을 작성할지 정합니다.
게시할 이미지 갯수 업로드시 들어갈 이미지 갯수입니다.

예시)게시할 갯수(아이디당):3 아이디당 3번씩 게시물이 등록됩니다.


7.대기시간 범위는 게시물을 업로드하고 다음 계정 넘어가기까지의 대기시간 입니다.

예시)최소시간:50 최대시간:90 50~90초 사이에 시간이 지난후 다음 계정으로 넘어갑니다.


※해쉬태그,내용은 필요없을시 미입력 하셔도 됩니다.

예시)
아이디1, 비밀번호1
아이디2, 비밀번호2

예시)image_folder 안 ex_image1.jpg, ex_image.png ...

폴더 안 이미지는 보이지 않습니다. 폴더 용량은 최대 50MB를 넘으면 안됩니다.

예시) 쓰하리1000명도전,쓰하리해요,쓰하리요청

4.내용 파일은 메모장에 “ / ” 로 구분해주시면 됩니다.

예시) 안녕하세요./잘부탁드립니다/오늘 엄청 덥네요

예시)게시할 갯수(아이디당):3 아이디당 3번씩 게시물이 등록됩니다.


6.대기시간 범위는 게시물을 업로드하고 다음 계정 넘어가기까지의 대기시간 입니다.

예시)최소시간:50 최대시간:90 50~90초 사이에 시간이 지난후 다음 계정으로 넘어갑니다.
